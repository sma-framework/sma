## 2. Display

**If `workspace_count` is 0:**

```
No workspaces found in ~/sma-workspaces/

Create one with:
  /sma-workspace --new --name my-workspace --repos repo1,repo2
```

Done.

**If workspaces exist:**

Display a table:

```
SMA Workspaces (~/sma-workspaces/)

| Name | Repos | Strategy | SMA Project |
|------|-------|----------|-------------|
| feature-a | 3 | worktree | Yes |
| feature-b | 2 | clone | No |

Manage:
  cd ~/sma-workspaces/<name>     # Enter a workspace
  /sma-workspace --remove <name>  # Remove a workspace
```

For each workspace, show:
- **Name** — directory name
- **Repos** — count from init data
- **Strategy** — from WORKSPACE.md
- **SMA Project** — whether `.planning/PROJECT.md` exists (Yes/No)
