**SMA — top commands**

```text
/sma:new-project           Initialize a project (greenfield)
/sma:onboard               Onboard an existing codebase (brownfield)
/sma:map-codebase          Refresh/map codebase intelligence
/sma:plan-phase <N>        Create a phase plan
/sma:execute-phase <N>     Execute a phase
/sma:progress              Where am I, what's next
/sma:quick                 Small ad-hoc task with SMA guarantees
/sma:fast "<task>"         Trivial inline task — no subagents
/sma:debug "<symptom>"     Persistent debug session (survives /clear)
/sma:capture               Save an idea / todo / note
/sma:ship <N>              Open a PR from a completed phase
```

More: `/sma:help` (default tour) · `/sma:help --full` (everything) · `/sma:help <topic>` (one section)
